# Word Clock JQuery plugin

[![Demo image](img_demo.png)]

## Instructions:

Include JS and CSS:

    <script src="wordclock.js"></script>
    <link rel="stylesheet" href="wordclock.css" />

Create a container:

    <div id="test"></div>

Do the magic:

    $('#test').wordClock();
